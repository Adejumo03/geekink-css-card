# GeekInk Profile Card

A pixel-perfect recreation of the GeekInk UI profile card component.

## Features
- **Responsive Flexbox Layout:** Perfectly centered on any screen size.
- **External CSS:** Keeps the project structure clean and scalable.
- **Hover Effects:** Interactive button states for better UX.

## How to Run
1. Save the `index.html` and `style.css` files in the same folder.
2. Open `index.html` in any modern web browser.

## Customization
- To change the background color, update the `.profile-card` background property in `style.css`.
- To change the profile image, replace the `src` attribute in the `<img>` tag.
